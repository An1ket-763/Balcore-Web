# Balcore Web

Balcore Web is the marketing site for Balcore's Avalanche-native liquidity infrastructure.

## Available Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run test`
- `npm run lint`

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## Notes

Static social metadata is configured in `index.html`, application assets live in `src/assets`, and `public/` contains files that must remain publicly served as-is.
